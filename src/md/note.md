## 第一章

### 1.1

**基本知识：**
    
    1.类名必须以字母开头，后面可以跟字母和数字任意组合，不能使用java中的保留字.首字母大写，多个单词采用驼峰命名
    2.源代码的文件名必须与公共类的名字相同
    3.每个语句后面必须以；结束
 
**数据类型:**
    
    整型: int 4byte
          short  2byte
          long   8byte
          byte  1byte
     浮点数：      
           float: 4字节（没有f默认为double类型）
           double: 8字节
    char:
            单引号：'A' 它表示编码值为65所对应的字符常量
            双引号：："A" 包含一个字符A的字符串
     bollean: true 和 false
     
 **常量：**
        
        finaly修饰，常量名大写（表示这个变量只能被赋值一次，一旦被赋值之后，就不能再更改了）
        
 **数值类型之间的转换**
        
        实线：表示无信息丢失的转换
        虚线：表示有信息丢失的转换
